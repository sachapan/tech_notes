### Changing CTRL-V and CTRL-C behavior on Windows terminal settings.json so vim keybindings still work for visual mode
  
    "actions": [
        {
            "command": "paste",
            "keys": "ctrl+shift+v"
        },
        {
            "command": {
                "action": "copy",
                "singleLine": false
            },
            "keys": "ctrl+shift+c"
        },
