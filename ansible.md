# Ansible stuff goes here

## Convert inventory ini to yaml or json

| Format | Command |
| --- | --- |
| yaml | `ansible-inventory -i inventory.ini -y --list > inventory.yaml` |
| json | `ansible-inventory -i inventory.ini --list > inventory.yaml` |
